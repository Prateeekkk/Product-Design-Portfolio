# Portfolio Design System

## Design Principles
- Light mode only
- Minimal but expressive
- Strong hierarchy, spacing, and alignment
- No unnecessary decoration
- Every element must serve a purpose

## Tone of Voice
- Clear, sharp, and human
- Slightly witty, dry humor
- Confident but not arrogant
- Avoid corporate jargon and generic phrases

## Copywriting Guidelines
- No "passionate designer" or clichés
- Use conversational tone
- Add small personality moments (microcopy, labels, hover text)
- Keep humor subtle and intentional

## Interaction Philosophy
- Micro-interactions over heavy animations
- Smooth transitions
- Feedback on hover and actions
- Make UI feel responsive and alive

## Layout System
- Grid-based structure
- Consistent spacing scale
- Clear visual anchors per section

## What to Avoid
- Random floating elements without meaning
- Overuse of gradients or colors
- Cluttered UI
- Copy that sounds like a resume
